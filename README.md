# gRPC-Cource

## Getting Started

### Installation

```shell
npm init -y

yarn add google-protobuf @grpc/grpc-js 
```

### Add Dev Dependencies

```shell
yarn add -D  grpc-tools ts-node typescript
```

> If you are on ARM Architecture then run the following
> ```shell 
> npm_config_target_arch=x64 yarn add -D  grpc-tools
>```
